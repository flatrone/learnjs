Деструктуризация массива

let [a, b] = arr

[a, , b] = arr - пропуск элементов

[obj.val1, obj.val2] = arr - Деструктуризация в свойства объект

обмен переменных

[a, b] = [b, a]

остаточные параметры

[a, b, ...rest] = arr - в rest запишется все остальное

значения по умольчанию

[a = "a", b = 'b'] = arr - если элемент будет отсутствовать

Деструктуризация объекта

let {var1, var2} = {var1: ..., var2: ...}

{name, surname, age} = user - если в объекте такие же названия свойств, то деструктуризация пройдет во все указанные свойства. Порядок не имеет значения

{name: n, age: a} = user - присвоить другую переменную

{a = 'a', b = 'b'} = obj - значения на случай их отсутствия

{a, b} = obj - если переменные объявлены где то в начале, а потом сделать такую деструктуризацию, то так работать не будет. JS будет считать {a, b} как блочный кусок кода. Можно обернуть все в ()

Вложенная деструктуризация

{key: {val1, val2}} = obj

параметры функции

function foo({a = "100", b: c}) {}

foo(obj)

foo({}) - если объекта нет
