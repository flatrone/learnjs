Статические методы

по сути это просто обычные функции объявленные внутри класса

по типу Array.isArray(), Object.keys()

          class MathHelper {
            // Статический метод для сложения двух чисел
            static add(a, b) {
              return a + b;
            }
          }
          
          // Вызываем статические методы непосредственно на классе
          console.log(MathHelper.add(5, 3)); // 8

к созданным объектам из класса эти методы не применимы. obj.add() - не сработает

статические свойства

по сути просто свойства класса

          class Article {
            static publisher = "Илья Кантор";
          }
          
          alert( Article.publisher ); // Илья Кантор

тоже самое что и

Article.publisher = "Илья Кантор";
