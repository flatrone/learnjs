Класс

класс - это как new function, только новое и продвинутое

напоминание: new function - используется для создания объектов

синтаксис

        class User {

            constructor(name) {
              this.name = name;
            }
          
            sayHi() {
              alert(this.name);
            }
          
          }
          
          // Использование:
          let user = new User("Иван");
          user.sayHi();
