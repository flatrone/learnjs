map

map - это объект, где ключем может быть что угодно

new Map() – создаёт коллекцию. map.set(key, value) – записывает по ключу key значение value. map.get(key) – возвращает значение по ключу или undefined, если ключ key отсутствует. map.has(key) – возвращает true, если ключ key присутствует в коллекции, иначе false. map.delete(key) – удаляет элемент (пару «ключ/значение») по ключу key. map.clear() – очищает коллекцию от всех элементов. map.size – возвращает текущее количество элементов.
