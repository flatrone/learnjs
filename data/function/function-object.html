лбъект функция

свойство name

все ниже одинаково присваивается имя foo.name

        function sayHi() {
            alert("Hi");
          }
          
        let sayHi = function() {
            alert("Hi");
        };

свойство length

показывает длинну принимаемых параметров

function foo(a, b) {}; foo.length //покажет 2

пользовательские свойства

        function foo(){
            foo.counter++
        }
        foo.counter = 0;
        foo()
        alert(foo.counter)

если объявить counter через let, это другое

let объявляет локальную переменную, а foo.counter задает свойство объекта функции
