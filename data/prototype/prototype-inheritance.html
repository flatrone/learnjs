Прототипное наследование

Прототипное наследование - создать новый объект на основе другого?

[[Prototype]]

это скрытое свойство объекта которое ссылается на другой объект(прототип, родительский объект)

Прототип даёт нам немного «магии». Когда мы хотим прочитать свойство из object, а оно отсутствует, JavaScript автоматически берёт его из прототипа. В программировании такой механизм называется «прототипным наследованием».

        let animal = {
            eats: true
          };
          
          let rabbit = {
            jumps: true
          };
          
          rabbit.__proto__ = animal;

__proto__ - устанавливает для объекта прототип

let rabbit = {__proto__: animal} - можно записать прототип в свойтва

__proto__ - уже устарело
