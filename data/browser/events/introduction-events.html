Введение в события

события мыши

  • click - левая кнопка
  • contextmenu - правая кнопка
  • mouseover / mouseout - когда мышь наводится \ покидает элемент
  • mousedown / mouseup - нажали / отпустили кнопку
  • mousemove - движение мыши

события на элементах управления

  • submit - отправка формы
  • focus - фокус на элементе, например нажатие на input

клавиатурные события

  • keydown / keyup - кнопку нажать / отпустить

обработчик событий

div onclick='alert("hi")' - событие через атрибуты в html

elem.onclick = function() {...} - через JS

доступ к элементу через this

div onclick='alert(this)'

скобки

при установке событий через атрибут, скобки нужны. а через JS не нужны

addEventListener / removeEventListener

elem.addEventListener(event, handler)

отиличие от onclick - через addEvent можно добавлять несколько обработчиков подряд

объект события(event)

elem.onclick = function(event) {alert(event.всякие свойства)}

некоторые свойства

event.type - тип события

event.currentTarget - элемент на котором сработал обработчик

event.clientX/Y - координаты курсора в момент клика

объект обработчик handleEvent
