Объекты

let obj = new Object() - синтаксис конструктор объекта

let obj = {} - синтаксис литерал объекта

delete obj.value - удаление

{"big value": true} - если свойство состоит из нескольких слов, то оно помещается в кавычки

константные объекты изменяемые, т.к. тип самой переменной объекта не меняется. константа защищает саму переменную, а не ее содержимое

"key" in obj - проверка свойства в объекте

вычисляемые свойства

let num = prompt('введите число', '5'); let obj = {[num]: 'ваше введенное число'} или obj[num] = 'ваше число' или obj{[num + "ваше"]: 'число'}
