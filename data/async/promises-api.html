promises Api

в классе Promise есть 6 методов

Promises.all

Запустить много промисов и дождаться, пока они все выполнятся

метод all - принимает массив промисов и возвращает новый массив

        Promise.all([
            new Promise(resolve => setTimeout(() => resolve(1), 3000)), // 1
            new Promise(resolve => setTimeout(() => resolve(2), 2000)), // 2
            new Promise(resolve => setTimeout(() => resolve(3), 1000))  // 3
            ]).then(alert); // когда все промисы выполнятся, результат будет 1,2,3
            // каждый промис даёт элемент массива

если хоть в одном возникнет ошибка, то отменяются все промисы

Promise.race

похож на .all, но из массива кто первый выполнится(ошибочный или успешный) тот и работает дальше, остальные мимо

Promise.any

похож на .race, но ждет УСПЕШНЫЙ результат, а не абы какой
